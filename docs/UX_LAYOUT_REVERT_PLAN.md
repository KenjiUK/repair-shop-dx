# UXレイアウト - 以前のデザインに戻す計画

## ユーザーの要求

1. **上部と下部に分ける**（現在の実装と同じ）
2. **下部は作業一覧だけ、左寄せで表示**（現在は2列レイアウトだが、1列の左寄せに戻す）
3. **写真は以前のサイズに戻す**（現在は`flex flex-col`で縦に並んでいるが、以前は横並びだった）
4. **★やフォルダ、プリンタは右上に表示**（現在は下部セクションの左側にあるが、これを右上に戻す）

---

## 理解した仕様

### 上部セクション
- **左側**: 車両画像（以前のサイズ、横並び、`flex flex-col`なし）
- **中央**: 顧客情報・車両情報
- **右側**: 
  - 星、フォルダ、プリンタアイコン（顧客名の横）
  - 受付/引渡しボタン（該当する場合）

### 下部セクション
- **全幅**: 作業一覧（左寄せで表示）
- アイコンボタンは表示しない

---

## 実装方針

### 1. 写真セクションを以前のサイズに戻す
- `flex flex-col`を削除
- 横並びのレイアウトに戻す
- アイコンボタンを削除

### 2. 星、フォルダ、プリンタを右上に戻す
- 顧客名の横に配置
- 以前の位置に戻す

### 3. 下部セクションを1列の左寄せに変更
- 2列レイアウトを削除
- 全幅で左寄せ表示
- アイコンボタンエリアを削除
