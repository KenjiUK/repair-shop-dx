## 1. 確認済み事項

### 1-1. 業務フローパターン

**重要な気づき**: 
- **パターンA（入庫してから見積もり）**: 入庫→見積もり→承認→作業（車検・12ヶ月点検など）
- **パターンB（事前にメール等で見積もり）**: メール等で見積もり→了承→入庫→作業（エンジンオイル交換、タイヤ交換・ローテーションなど）

**タイヤ交換・ローテーション**: ✅ **パターンB（事前にメール等で見積もり→了承→入庫→作業）**

**設計決定**:
- 予約時にメール等で見積もりを送信
- 顧客が了承
- 入庫日を確定
- **タイヤ持ち込みの場合もある**（顧客が自分でタイヤを用意する場合）
- 入庫当日は作業のみ実施

### 1-2. 業務フロー

**質問1**: タイヤ交換とローテーションは、同じ作業として扱いますか？それとも別々の作業として扱いますか？

**回答**: ✅ **別々の作業として扱う**

**設計決定**:
- タイヤ交換とローテーションは**別々の入庫区分**として管理
- それぞれ独立した作業フローを持つ
- ただし、同じ画面設計・データモデルを共有可能

**質問2**: タイヤ交換・ローテーションの所要時間はどのくらいですか？

**回答**: ✅ **2時間**

**設計決定**:
- 標準: 2時間で完了（即日完了）
- タイヤ交換の場合: タイヤの用意が必要（事前に発注・用意済み）
- タイヤローテーションの場合: 既存のタイヤを使用するため、即日対応可能

### 1-3. 費用

**質問3**: タイヤ交換とローテーションの費用はいくらですか？

**回答**: ✅ 
- **タイヤ交換（4本）**: ¥30,800（バランス調整込、16インチ迄、タイヤ代別）
- **タイヤローテーション**: 工賃のみ（バランス調整込、16インチ迄、タイヤ代別）- 金額は「あたりでいれておいて」

**設計決定**:
- **タイヤ交換（4本）**: ¥30,800（工賃のみ、タイヤ代別）
  - バランス調整込
  - 16インチ迄
  - タイヤ代は別途必要（顧客が選択したタイヤ）
- **タイヤローテーション**: 工賃のみ（金額は要確認・設定）
  - バランス調整込
  - 16インチ迄
  - タイヤ代は不要（既存のタイヤを使用）
- 法定費用はなし（工賃のみ）

### 1-4. 検査項目

**質問5**: タイヤ交換・ローテーション時の簡易検査項目は、どのように入力しますか？

**回答**: ✅ **特にないので輸入車の文脈の推奨でベストを提案**

**推奨検査項目（提案・採用）**:
1. **タイヤの状態確認**
   - タイヤの溝の深さ（前後左右）
   - タイヤの摩耗状態（偏摩耗、片減りなど）
   - タイヤの損傷（キズ、パンク痕、バルジなど）
   - タイヤの製造年（DOT表示）
2. **空気圧の確認**
   - 前後左右の空気圧
   - 推奨空気圧との比較
   - 空気圧の均一性
3. **ホイールの状態確認**
   - ホイールの損傷（クラック、曲がりなど）
   - ホイールナットの緩み
   - ホイールバランスの状態

**設計決定**:
- 簡易検査項目をリスト形式で表示
- 各項目の状態を選択（正常/注意/要交換/要修理）
- 必要に応じて写真撮影
- 輸入車の文脈を考慮した検査項目を採用

### 1-5. 測定値の入力

**質問6**: 測定値の入力は必要ですか？

**回答**: ✅ **特にないので輸入車の文脈の推奨でベストを提案**

**推奨案（業務実態を考慮・輸入車の文脈）**: ✅ **採用**
- **タイヤ溝深さ**: 前後左右の溝深さを測定・記録（mm単位）
  - 法定基準: 1.6mm以上
  - 推奨基準: 3mm以上（輸入車の高性能タイヤの場合）
- **空気圧**: 前後左右の空気圧を測定・記録（kPaまたはpsi単位）
  - 推奨空気圧は車両情報から自動取得
  - 実際の空気圧と推奨空気圧を比較表示
- **その他の測定値**: 不要（簡易検査のみ）

**設計決定**:
- 測定値入力画面を実装
- タイヤ溝深さ（前後左右）を入力
- 空気圧（前後左右）を入力
- 推奨値との比較を自動表示

### 1-6. 写真・動画

**質問7**: 写真・動画の撮影は必要ですか？

**回答**: ✅ **はい**

**設計決定**:
- **Before写真**: 交換前のタイヤの状態を撮影
  - タイヤ交換の場合: 交換前のタイヤ全体（4本）
  - ローテーションの場合: ローテーション前のタイヤ位置
- **After写真**: 交換後のタイヤの状態を撮影
  - タイヤ交換の場合: 交換後のタイヤ全体（4本）
  - ローテーションの場合: ローテーション後のタイヤ位置
- 動画は不要（写真のみ）
- 顧客への説明材料として有効

### 1-7. 部品リストアップ

**質問8**: 診断時の部品リストアップは必要ですか？

**回答**: ✅ **記録は費用だが部品リストアップは不要**

**設計決定**:
- **タイヤ交換の場合**:
  - 交換するタイヤの種類・サイズ・メーカーは**作業記録として記録**（費用計算のため）
  - ただし、診断時の部品リストアップ機能は不要（予約時に既に確定している）
- **タイヤローテーションの場合**:
  - 部品リストアップは不要（既存のタイヤを使用）
- 診断画面には部品リストアップセクションを表示しない

### 1-8. 見積もり・承認フロー

**質問9**: 見積もりの承認フローはどうしますか？

**回答**: ✅ **事前に決まりますね。他のメンテも事前に価格が決まってるので承認不要**

**設計決定**:
- 見積もり画面は不要（予約時に実施が確定している）
- 承認フローも不要（事前にメール等で了承済み）
- 入庫当日は作業のみ実施
- エンジンオイル交換と同様のパターンB

### 1-9. 作業記録

**質問10**: 作業記録はどうすればいいですか？

**回答**: ✅ **特にないので輸入車の文脈の推奨でベストを提案**

**推奨案（提案・採用）**:
- **タイヤ交換の場合**:
  - 交換したタイヤの種類・サイズ・メーカー（例: ブリヂストン 225/45R17）
  - 交換位置（前後左右）
  - バランス調整結果（各ホイールのバランス値）
  - 空気圧設定値（前後左右）
  - Before/After写真
  - タイヤ溝深さ（交換前の記録）
- **タイヤローテーションの場合**:
  - ローテーション位置（例: 前左→後右、前右→後左、後左→前左、後右→前右）
  - バランス調整結果（各ホイールのバランス値）
  - 空気圧設定値（前後左右）
  - Before/After写真
  - タイヤ溝深さ（ローテーション前後の記録）

**設計決定**:
- 作業画面で作業記録を入力
- タイヤ交換の場合: 交換したタイヤの種類・サイズ・メーカーを記録
- タイヤローテーションの場合: ローテーション位置を記録
- バランス調整結果を記録
- 空気圧設定値を記録
- Before/After写真を撮影
- タイヤ溝深さを記録（作業前後）

### 1-10. PDFテンプレート

**質問11**: タイヤ交換・ローテーション用のPDFテンプレートはありますか？

**回答**: ✅ **PDFテンプレートはない。PDF生成は不要**

**設計決定**:
- タイヤ交換・ローテーションではPDF生成は不要
- PDFで出すのは車検と12ヶ月点検のみ（決まったフォーマット）
- 作業記録はシステム内で管理し、顧客レポート画面で表示

### 1-11. データソース

**質問12**: 車両情報や費用情報の取得元は車検・12ヶ月点検と同じですか？

**回答**: ✅ **全部一緒です**

**設計決定**:
- Google Sheets「【DB】車両マスタ」から車両情報を取得
- 工賃は固定価格（タイヤ交換: ¥30,800、ローテーション: 要確認）または基幹システムから取得
- タイヤ代は顧客が選択したタイヤに応じて変動（タイヤ交換の場合）

## 2. 確定した仕様サマリー

### 2-1. 業務フロー（パターンB）

**タイヤ交換の場合**:
```
1. 予約時: メール等で見積もりを送信
   - タイヤ代（顧客が選択したタイヤ）
   - 工賃: ¥30,800（バランス調整込、16インチ迄、タイヤ代別）
2. 顧客が了承
3. タイヤを発注・用意（またはタイヤ持ち込みの場合もある）
4. 入庫日を確定
5. 入庫当日:
   - フロントスタッフが受付
   - 整備士が簡易検査を実施
     - タイヤの状態確認
     - 空気圧の確認
     - タイヤ溝深さの測定
     - 交換前のタイヤの写真を撮影（Before）
   - 作業を実施（2時間）
     - タイヤ交換（4本）
     - バランス調整
     - 空気圧調整
     - 交換後のタイヤの写真を撮影（After）
     - 作業記録を入力
   - 作業完了
   - フロントスタッフが引渡し・説明
6. 出庫完了
```

**タイヤローテーションの場合**:
```
1. 予約時: メール等で見積もりを送信
   - 工賃: ¥X,XXX（バランス調整込、16インチ迄、タイヤ代別）
2. 顧客が了承
3. 入庫日を確定
4. 入庫当日:
   - フロントスタッフが受付
   - 整備士が簡易検査を実施
     - タイヤの状態確認
     - 空気圧の確認
     - タイヤ溝深さの測定
     - ローテーション前のタイヤ位置の写真を撮影（Before）
   - 作業を実施（2時間）
     - タイヤローテーション（位置変更）
     - バランス調整
     - 空気圧調整
     - ローテーション後のタイヤ位置の写真を撮影（After）
     - 作業記録を入力
   - 作業完了
   - フロントスタッフが引渡し・説明
5. 出庫完了
```

## 3. エンジンオイル交換との比較

| 項目 | エンジンオイル交換 | タイヤ交換 | タイヤローテーション |
|------|------------------|-----------|-------------------|
| **業務フローパターン** | パターンB（事前見積→了承→入庫→作業） | パターンB（事前見積→了承→入庫→作業） | パターンB（事前見積→了承→入庫→作業） |
| 所要時間 | 1時間以内 | **2時間** | **2時間** |
| 検査項目数 | 簡易検査（3項目） | **簡易検査（3カテゴリ）** | **簡易検査（3カテゴリ）** |

| 部品リストアップ | 基本不要（イレギュラーで見つけた場合のみ） | **記録は費用だが部品リストアップ機能は不要** | **不要** |
| 見積もりフロー | 不要（予約時に実施確定） | **不要（予約時に実施確定）** | **不要（予約時に実施確定）** |
| 承認フロー | 不要（事前に了承済み） | **不要（事前に了承済み）** | **不要（事前に了承済み）** |
