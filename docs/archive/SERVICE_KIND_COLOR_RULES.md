# 入庫区分別アイコンの色ルール

## 📋 概要

入庫区分別のアイコンの色は、**カテゴリー色システム**に基づいて統一されています。
これは、ステータス色（入庫待ち、作業待ちなど）とは**別の色体系**です。

---

## 🎨 カテゴリー色システム

### 色の分類ルール

入庫区分は、作業の性質に応じて以下の6つのカテゴリーに分類されています：

| カテゴリー | 色 | 意味 | 該当する入庫区分 |
|-----------|-----|------|----------------|
| **点検・検査系** | Cyan (青緑) | 定期点検や検査作業 | 車検、12ヵ月点検 |
| **メンテナンス系** | Emerald (緑) | 日常的なメンテナンス作業 | エンジンオイル交換、タイヤ交換・ローテーション |
| **修理・整備系** | Orange (オレンジ) | 故障や不具合の修理 | 修理・整備 |
| **診断・トラブル系** | Rose (ローズ) | 故障診断やトラブル調査 | 故障診断 |
| **カスタマイズ・特殊作業系** | Violet (紫) | カスタマイズや特殊な作業 | チューニング、パーツ取付、コーティング、レストア、板金・塗装 |
| **その他** | Slate (グレー) | 上記に該当しない作業 | その他、その他のメンテナンス |

### 色の詳細

```typescript
// 点検・検査系（Cyan）
車検: text-cyan-600
12ヵ月点検: text-cyan-600

// メンテナンス系（Emerald）
エンジンオイル交換: text-emerald-600
タイヤ交換・ローテーション: text-emerald-600

// 修理・整備系（Orange）
修理・整備: text-orange-700

// 診断・トラブル系（Rose）
故障診断: text-rose-600

// カスタマイズ・特殊作業系（Violet）
チューニング: text-violet-700
パーツ取付: text-violet-700
コーティング: text-violet-700
レストア: text-violet-700
板金・塗装: text-violet-700

// その他（Slate）
その他: text-slate-700
その他のメンテナンス: text-slate-700
```

## ❓ なぜ同じ色があるのか？

### 理由1: カテゴリーによる分類

同じ色が使われているのは、**同じカテゴリーに属する入庫区分**だからです。

例えば：
- **Violet (紫)** が5つの入庫区分で使われているのは、これらがすべて「カスタマイズ・特殊作業系」に分類されるためです
- **Cyan (青緑)** が2つの入庫区分で使われているのは、これらが「点検・検査系」に分類されるためです

### 理由2: 視覚的なグループ化

同じ色を使うことで、ユーザーは：
- 同じカテゴリーの作業を**視覚的にグループ化**して認識できる
- 作業の性質を**色で直感的に理解**できる
- フィルターやサマリーカードで**関連する作業を見つけやすくなる**

### 理由3: ステータス色との分離

入庫区分の色は、ステータス色（入庫待ち、作業待ちなど）とは**別の色体系**です。

- **ステータス色**: 作業の進行状況を表す（例: 入庫待ち=赤、作業待ち=緑）
- **カテゴリー色**: 作業の性質を表す（例: 点検系=青緑、メンテナンス系=緑）

これにより、ユーザーは：
- ステータスとカテゴリーを**同時に認識**できる
- 色の意味が**混同されない**
