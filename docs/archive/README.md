# アーカイブドキュメント

このフォルダには、実装完了後や統合後に整理された過去のドキュメントを保持しています。

## アーカイブの目的

- **設計の経緯の記録**: なぜその設計にしたのか、判断理由を記録
- **将来の参照**: 類似機能の実装時に参考にする
- **プロジェクトの変遷**: システムの進化過程を記録
