# UX修正: ボタン配置の改善

## 問題点

現在の実装では、単発作業の場合、右上に「診断を開始」「作業を開始」などのボタンが表示されている。これらは作業単位の操作なので、右上（車両単位の操作の位置）に置くのはUX的にベストプラクティスではない。

## 修正方針

### ボタンの配置ルール

1. **右上に表示するボタン（車両単位の操作）**:
   - 受付を開始（入庫待ち）
   - 引渡しを開始（出庫待ち）

2. **作業カード内に表示するボタン（作業単位の操作）**:
   - 診断を開始（入庫済み）
   - 見積を開始（見積作成待ち）
   - 作業を開始（作業待ち）

### 実装方針

- **単発作業の場合でも、作業カードを表示**
- 作業カード内にアクションボタンを配置
- 右上には車両単位の操作のみを表示
