## 1. 対象となるメンテナンス系メニュー一覧

### 1-1. 一般的な消耗品・メンテナンス系メニュー

| **エアフィルター交換** | 15-30分 | 1-2年 | エンジンの吸気フィルター交換 |
| **クーラント（冷却水）交換** | 1-2時間 | 2-3年 | エンジンの冷却システム |

| **ワイパー交換** | 15-30分 | 1-2年 | フロント・リアワイパー |
| **ブレーキパッド交換** | 1-2時間 | 3-5年 | 安全に関わる重要なメンテナンス |

## 2. 業務フローパターン分析

### 2-1. 共通の業務フローパターン

すべてのメンテナンス系メニューは、**パターンB（事前にメール等で見積もり→了承→入庫→作業）** を採用します。

**理由**:
- 部品の在庫確認が必要
- 事前に費用を提示して顧客の了承を得る
- 入庫当日は作業のみ実施（スムーズな対応）

### 2-2. 業務フロー（汎用的なフロー）

#### Phase 0: 予約・事前準備
- **顧客の行動**: 定期メンテナンスの予約、または不具合を感じた時の予約
- **スタッフの行動**: 
  - 前回交換日・走行距離の確認
  - 部品の在庫確認
  - 車両情報の確認（適合部品の特定）
- **システム要件**: 
  - 前回交換履歴の表示
  - 部品在庫の確認
  - 車両情報からの適合部品の自動提案

#### Phase 1: 受付
- **所要時間**: 約3-5分
- **確認事項**: 
  - 走行距離
  - 前回交換日（該当する場合）
  - 不具合の有無（該当する場合）
  - 顧客の要望（部品のグレードなど）
- **記録内容**: 
  - 走行距離
  - 顧客の要望事項

#### Phase 2: 診断・検査（簡易）
- **所要時間**: 約10-30分（メニューによって異なる）
- **検査項目**: 
  - **交換対象部品の状態確認**
    - バッテリー交換 → バッテリー電圧、端子の状態
    - ブレーキフルード交換 → フルードの色、量、汚れ具合
    - エアフィルター交換 → フィルターの汚れ具合
    - クーラント交換 → 冷却水の色、量、汚れ具合
    - スパークプラグ交換 → プラグの状態、ギャップ
    - ベルト類交換 → ベルトのひび割れ、張り具合
    - ワイパー交換 → ワイパーの劣化状態
    - ブレーキパッド交換 → パッドの残量、ローターの状態
    - ブレーキローター交換 → ローターの摩耗、歪み
  - **関連箇所の簡易点検**（必要に応じて）
- **記録内容**: 
  - 交換対象部品の状態
  - 測定値（電圧、残量、摩耗量など）
  - 写真（必要に応じて）

#### Phase 3: 見積作成（簡易）
- **所要時間**: 約5-10分
- **作業内容**: 
  - 部品代・工賃の見積もり
  - 交換方法の説明（必要に応じて）
- **記録内容**: 
  - 部品代・工賃の内訳
  - 交換方法の説明

#### Phase 4: 顧客承認（即決が多い）
- **所要時間**: 通常即決（事前にメール等で了承済みの場合が多い）
- **顧客の行動**: 
  - 見積内容の確認
  - 承認
- **システム要件**: 
  - 簡易な承認フロー

#### Phase 5: 作業実施
- **所要時間**: メニューによって異なる（15分〜3時間）
- **作業内容**: 
  - 交換作業の実施
  - 動作確認
  - 関連箇所の点検（必要に応じて）
- **記録内容**: 
  - 交換した部品の種類・型番
  - 交換部品の記録
  - Before/After写真（必要に応じて）
  - 動作確認の結果

#### Phase 6: 引渡・説明
- **所要時間**: 約5-10分
- **作業内容**: 
  - 交換内容の説明
  - 次回交換時期の案内
  - 注意事項の説明
- **記録内容**: 
  - 引渡日時
  - 次回交換予定日（該当する場合）

## 3. メニュー別の特徴・違い

### 3-1. バッテリー交換

**特徴**:
- **所要時間**: 30分〜1時間
- **検査項目**: バッテリー電圧、端子の状態、充電状態
- **測定値**: 電圧（V）、CCA（Cold Cranking Amps）
- **写真**: Before/After写真（端子の状態など）
- **部品**: バッテリーの型番、容量、CCA値
- **注意事項**: 車両の電装品への影響、適合バッテリーの確認

### 3-2. ブレーキフルード交換

**特徴**:
- **所要時間**: 30分〜1時間
- **検査項目**: フルードの色、量、汚れ具合、水分含有率
- **測定値**: 水分含有率（%）、沸点（℃）
- **写真**: Before/After写真（フルードの色の変化）
- **部品**: フルードの種類（DOT3/DOT4/DOT5.1）、容量
- **注意事項**: 安全に関わる重要なメンテナンス、定期的な交換が必要

### 3-3. エアフィルター交換

**特徴**:
- **所要時間**: 15-30分
- **検査項目**: フィルターの汚れ具合、目詰まり
- **測定値**: なし（目視確認）
- **写真**: Before/After写真（フィルターの汚れ具合）
- **部品**: エアフィルターの型番
- **注意事項**: エンジンの性能に影響、定期的な交換が必要

### 3-4. クーラント（冷却水）交換

**特徴**:
- **所要時間**: 1-2時間
- **検査項目**: 冷却水の色、量、汚れ具合、凍結防止性能
- **測定値**: 凍結防止性能（℃）、pH値
- **写真**: Before/After写真（冷却水の色の変化）
- **部品**: クーラントの種類、容量
- **注意事項**: エンジンのオーバーヒート防止、定期的な交換が必要

### 3-5. スパークプラグ交換

**特徴**:
- **所要時間**: 1-2時間
- **検査項目**: プラグの状態、ギャップ、焼け具合
- **測定値**: ギャップ（mm）、抵抗値（Ω）
- **写真**: Before/After写真（プラグの状態）
- **部品**: スパークプラグの型番、本数
- **注意事項**: エンジンの点火性能に影響、定期的な交換が必要

### 3-6. ベルト類交換

**特徴**:
- **所要時間**: 1-3時間（ベルトの種類による）
- **検査項目**: ベルトのひび割れ、張り具合、摩耗
- **測定値**: 張力（N）、ひび割れの本数
- **写真**: Before/After写真（ベルトの状態）
- **部品**: ベルトの型番、種類（ファンベルト/タイミングベルトなど）
- **注意事項**: タイミングベルトはエンジンの破損に関わる重要な部品

### 3-7. ワイパー交換

**特徴**:
- **所要時間**: 15-30分
- **検査項目**: ワイパーの劣化状態、拭き取り性能
- **測定値**: なし（目視確認）
- **写真**: Before/After写真（ワイパーの状態）
- **部品**: ワイパーの型番、サイズ（フロント/リア）
- **注意事項**: 視認性に関わる重要な部品、定期的な交換が必要

### 3-8. ブレーキパッド交換

**特徴**:
- **所要時間**: 1-2時間
- **検査項目**: パッドの残量、ローターの状態、異音の有無
- **測定値**: パッド残量（mm）、ローターの厚み（mm）
- **写真**: Before/After写真（パッドの残量、ローターの状態）
- **部品**: ブレーキパッドの型番、セット数
- **注意事項**: 安全に関わる重要なメンテナンス、定期的な交換が必要

### 3-9. ブレーキローター交換

**特徴**:
- **所要時間**: 1-2時間
- **検査項目**: ローターの摩耗、歪み、スコアリング
- **測定値**: ローターの厚み（mm）、歪み（mm）
- **写真**: Before/After写真（ローターの状態）
- **部品**: ブレーキローターの型番、セット数
- **注意事項**: 安全に関わる重要なメンテナンス、ブレーキパッドとセットで交換されることが多い

### 3-10. オイルフィルター交換（単独）

**特徴**:
- **所要時間**: 15-30分
- **検査項目**: フィルターの汚れ具合、目詰まり
- **測定値**: なし（目視確認）
- **写真**: Before/After写真（フィルターの汚れ具合）
- **部品**: オイルフィルターの型番
- **注意事項**: オイル交換とセットで実施されることが多い

### 3-11. キャビンフィルター交換

**特徴**:
- **所要時間**: 15-30分
- **検査項目**: フィルターの汚れ具合、目詰まり、臭い
- **測定値**: なし（目視確認）
- **写真**: Before/After写真（フィルターの汚れ具合）
- **部品**: キャビンフィルターの型番
- **注意事項**: エアコンの性能、車内の空気質に関わる

### 3-12. ラジエーターキャップ交換

**特徴**:
- **所要時間**: 5-10分
- **検査項目**: キャップの劣化、シールの状態
- **測定値**: 圧力（kPa）
- **写真**: Before/After写真（キャップの状態）
- **部品**: ラジエーターキャップの型番
- **注意事項**: 簡易な交換作業、冷却システムの圧力管理に関わる

## 4. 汎用的なシステム要件（あるべき姿）

### 4-1. 診断画面での要件

#### 共通項目
- **交換対象部品の状態確認**: メニューに応じた検査項目を表示
- **測定値の入力**: メニューに応じた測定値の入力フィールド
- **写真撮影**: Before/After写真（必要に応じて）
- **コメント入力**: 整備士の所見（音声入力対応）

#### メニュー別のカスタマイズ
- **検査項目の表示**: 選択されたメニューに応じて、関連する検査項目を自動表示
- **測定値フィールド**: メニューに応じた測定値フィールドを動的に表示
- **写真撮影の必須/任意**: メニューに応じて、写真撮影を必須または任意に設定

### 4-2. 見積画面での要件

#### 共通項目
- **部品代・工賃の見積もり**: 明確な内訳表示
- **交換方法の説明**: 必要に応じて交換方法を説明
- **写真・コメントの紐付け**: 各見積項目に証拠写真を紐付け

#### メニュー別のカスタマイズ
- **部品の選択**: 車両情報から適合部品を自動提案
- **グレードの選択**: 複数のグレードがある場合、選択可能に
- **セット交換の提案**: 関連部品のセット交換を提案（例: ブレーキパッド+ローター）

### 4-3. 作業画面での要件

#### 共通項目
- **作業項目の表示**: 承認された項目のみ表示
- **Before/After写真**: 交換部品の証拠写真
- **交換部品の記録**: 交換した部品の種類・型番・数量
- **動作確認**: 作業後の動作確認結果

#### メニュー別のカスタマイズ
- **作業手順の表示**: メニューに応じた作業手順を表示
- **注意事項の表示**: メニューに応じた注意事項を表示
- **関連作業の提案**: 関連するメンテナンスの提案（例: ブレーキパッド交換時にローターも確認）

### 4-4. データモデル

#### 共通データ構造
```typescript
interface MaintenanceJob {
  // 基本情報
  id: string;
  serviceKind: "その他のメンテナンス";
  maintenanceType: MaintenanceType; // バッテリー交換、ブレーキフルード交換など
  
  // 診断情報
  diagnosis: {
    inspectionItems: InspectionItem[]; // 検査項目
    measurements: Record<string, number>; // 測定値
    photos: Photo[]; // 写真
    comments: string; // コメント
  };
  
  // 見積情報
  estimate: {
    parts: Part[]; // 部品
    labor: number; // 工賃
    total: number; // 合計
  };
  
  // 作業情報
  work: {
    replacedParts: ReplacedPart[]; // 交換部品
    photos: Photo[]; // Before/After写真
    workNotes: string; // 作業メモ
  };
}
```
