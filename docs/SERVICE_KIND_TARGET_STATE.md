# 入庫区分別「あるべき姿」設定書

## 目的

本ドキュメントは、各入庫区分ごとに「あるべき姿」を設定し、システム設計の指針とするための資料です。

## 設計原則

1. **入庫区分ごとの最適化**: 一律のプロセスではなく、各入庫区分に最適化されたプロセス
2. **所要日数の柔軟性**: 一日で終わる作業と複数日にまたがる作業の両方に対応
3. **検査項目のカスタマイズ**: 各入庫区分に応じた検査項目の表示
4. **記録内容の最適化**: 各入庫区分に必要な情報のみを記録

---

## 1. 車検（法定24ヶ月点検）

### 1-1. あるべき姿

#### プロセス
```
受付 → 診断（詳細検査） → 見積（法定/任意の区分） → 顧客承認 → 作業（法定点検+承認作業） → 引渡
```

#### 所要日数
- **標準**: 1日（午前入庫→午後出庫）
- **複数日**: 部品調達が必要な場合のみ

#### 検査項目
- **法定項目（必須）**: 100項目以上の点検項目を自動表示
- **任意項目（推奨）**: 顧客の要望に応じて追加

#### 記録内容
- **診断**: 各項目の状態、測定値、写真、コメント
- **見積**: 法定費用、部品代・工賃、写真・コメントの紐付け
- **作業**: 実施した作業、Before/After写真、交換部品

#### 画面要件
- **診断画面**: 車検用の100項目以上のチェックリスト、測定値入力、法定/任意の区分表示
- **見積画面**: 法定費用の自動計算、部品代・工賃の入力、写真・コメントの紐付け
- **作業画面**: 承認された項目のみ表示、Before/After写真、車検証更新の記録

## 2. 故障診断

### 2-1. あるべき姿

#### プロセス
```
受付 → 診断（原因特定） → 見積（原因説明+修理提案） → 顧客承認 → 作業（修理実施） → 引渡
```

#### 所要日数
- **標準**: 1日（原因特定が容易な場合）
- **複数日**: 原因特定が困難な場合、部品調達が必要な場合

#### 検査項目
- **症状に応じた検査**: 不具合内容から関連する検査項目を自動表示
- **関連箇所の検査**: 原因の可能性がある箇所

#### 記録内容
- **診断**: 検査結果、測定値、写真・動画、原因の推測、緊急度
- **見積**: 原因の説明、修理方法の提案、費用見積もり、緊急度の表示
- **作業**: 実施した修理、Before/After写真、交換部品、動作確認結果

#### 画面要件
- **診断画面**: 症状に応じた検査項目の自動表示、動画撮影、測定値入力、原因の推測、緊急度の設定
- **見積画面**: 原因の明確な説明、修理方法の選択肢、緊急度の表示
- **作業画面**: 承認された修理項目のみ表示、Before/After写真、動作確認の記録

## 5. エンジンオイル交換

### 5-1. あるべき姿

#### プロセス
```
受付 → 診断（簡易検査） → 見積（オイル代・工賃） → 顧客承認（即決が多い） → 作業（オイル交換） → 引渡
```

#### 所要日数
- **標準**: 1時間以内（即日完了）

#### 検査項目
- **簡易検査**: オイルの状態確認、オイルフィルターの確認、エアフィルターの確認

#### 記録内容
- **診断**: オイルの状態、写真（必要に応じて）
- **見積**: オイル代・工賃の内訳
- **作業**: 交換したオイルの種類・量、交換部品の記録

#### 画面要件
- **診断画面**: 簡易検査項目（オイル関連）、写真撮影（必要に応じて）
- **見積画面**: 簡易見積（オイル代・工賃のみ）
- **作業画面**: 簡易作業記録（交換内容のみ）

#### 特別要件
- **承認フロー**: 簡易な承認フロー（即決が多い）
- **所要時間**: 1時間以内で完了するため、プロセスを簡略化

## 共通要件

### プロセスの柔軟化

1. **即日完了作業**: エンジンオイル交換、タイヤ交換など
   - 診断→見積→承認→作業→引渡を1時間以内で完了
   - 承認フローを簡略化

2. **1日完了作業**: 車検、12ヵ月点検、修理・整備など
   - 午前入庫→午後出庫
   - 標準的なプロセス

3. **複数日作業**: レストア、チューニングなど
   - 進捗管理が必要
   - 作業日記的な記録

### 検査項目のマスタ化

- 各入庫区分ごとの検査項目をマスタデータとして管理
- 入庫区分に応じて、自動的に検査項目を表示

### 画面のカスタマイズ

- 各入庫区分に応じて、画面の表示内容をカスタマイズ
- 不要な項目は非表示、必要な項目のみ表示
