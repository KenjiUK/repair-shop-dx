# アクションボタンの色ルール

## 現在の実装ルール（2025-01-XX更新）

### 1. アイコン色
- すべてのアクションボタンのアイコンは`text-white`（白）で統一
- ボタンの背景色でステージを視覚的に区別

### 2. ボタン背景色（ステージ別）
ボタンの背景色はステージに応じて異なる色を使用します。

| ステージ | 背景色 | ホバー色 | 意味 |
|---------|--------|---------|------|
| 入庫待ち | `bg-slate-600` | `hover:bg-slate-700` | 初期状態 |
| 入庫済み | `bg-blue-600` | `hover:bg-blue-700` | 診断開始 |
| 見積作成待ち | `bg-orange-600` | `hover:bg-orange-700` | 見積作成 |
| 作業待ち | `bg-emerald-600` | `hover:bg-emerald-700` | 作業開始 |
| 出庫待ち | `bg-violet-600` | `hover:bg-violet-700` | 引渡し |

### 3. ボタンサイズ（優先度別）
| 優先度 | サイズ | Tailwindクラス | 使用ステージ |
|--------|--------|----------------|------------|
| high | 48px | `h-12` | 入庫待ち、入庫済み、見積作成待ち、作業待ち |
| medium | 40px | `h-10` | 出庫待ち |

### 4. アイコンサイズ（優先度別）
| 優先度 | サイズ | Tailwindクラス | 備考 |
|--------|--------|----------------|------|
| high | 20px | `h-5 w-5` | PC表示 |
| medium | 16px | `h-4 w-4` | PC表示（medium priorityの場合も`h-5 w-5`を使用） |

注: モバイル表示では`h-10`のボタンに`h-4 w-4`のアイコンを使用

### 5. ステージ別のアクション詳細
| ステージ | ラベル | アイコン | アイコン色 | ボタン背景色 | ホバー色 | ボタン高さ | 優先度 |
|---------|--------|---------|-----------|------------|---------|-----------|--------|
| 入庫待ち | 受付を開始 | Key | `text-white` | `bg-slate-600` | `hover:bg-slate-700` | `h-12` | high |
| 入庫済み | 診断を開始 | Activity | `text-white` | `bg-blue-600` | `hover:bg-blue-700` | `h-12` | high |
| 見積作成待ち | 見積を開始 | FileText | `text-white` | `bg-orange-600` | `hover:bg-orange-700` | `h-12` | high |
| 作業待ち | 作業を開始 | Wrench | `text-white` | `bg-emerald-600` | `hover:bg-emerald-700` | `h-12` | high |
| 出庫待ち | 引渡しを開始 | Car | `text-white` | `bg-violet-600` | `hover:bg-violet-700` | `h-10` | medium |
| 見積提示済み | - | - | - | - | - | - | none |
| 出庫済み | - | - | - | - | - | - | none |

## 実装状況

✅ **実装済み**: ボタンの背景色はステージに応じて適切に色分けされており、一見してどのステージか分かるように設計されています。

## デザインの意図

- **ステージ別の色分け**: 各ステージ（入庫待ち、診断待ち、見積作成待ち、作業待ち、引渡待ち）を色で視覚的に区別
- **優先度によるサイズ調整**: high priorityは`h-12`（48px）、medium priorityは`h-10`（40px）で重要度を表現
- **一貫性**: アイコンはすべて白で統一し、背景色で情報を伝達
