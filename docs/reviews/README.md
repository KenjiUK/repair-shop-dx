# UXテストレビュー提出先

このディレクトリには、各役割のユーザーから提出されたUXテストレビューを保存します。

---

## ファイル命名規則

**形式:** `UX_TESTING_REVIEW_[役割]_[名前]_YYYYMMDD.md`

**例:**
- `UX_TESTING_REVIEW_Receptionist_山田太郎_20251222.md`
- `UX_TESTING_REVIEW_Mechanic_佐藤花子_20251222.md`
- `UX_TESTING_REVIEW_Admin_鈴木一郎_20251222.md`
- `UX_TESTING_REVIEW_Manager_高橋美咲_20251222.md`

---

## 提出方法

1. レビューフォーム（[`../UX_TESTING_REVIEW_FORM.md`](../UX_TESTING_REVIEW_FORM.md)）に記入
2. 上記の命名規則に従ってファイル名を設定
3. このディレクトリに保存
