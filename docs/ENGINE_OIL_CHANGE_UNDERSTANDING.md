# エンジンオイル交換 業務理解メモ

## ドキュメント情報

- **作成日**: 2025-01-XX
- **バージョン**: 1.0
- **対象**: エンジンオイル交換の業務理解
- **目的**: 車検・12ヶ月点検と同様に、エンジンオイル交換の業務フローを想像し、仕様を確定する

---

## 1. エンジンオイル交換の基本情報

### 1-1. 業務の特徴

- **所要時間**: 1時間以内（即日完了）
- **頻度**: 定期的なメンテナンス（走行距離や期間に応じて）
- **作業内容**: エンジンオイルとオイルフィルターの交換
- **簡易検査**: オイルの状態確認、オイルフィルターの確認、エアフィルターの確認

### 1-2. 業務フローの特徴（推測）

- **受付から完了まで**: 1時間以内で完了するため、プロセスを簡略化
- **承認フロー**: 簡易な承認フロー（即決が多い）
- **記録内容**: 交換したオイルの種類・量、交換部品の記録

## 2. 確認済み事項

### 2-1. 業務フローについて

**質問1**: エンジンオイル交換の所要時間はどのくらいですか？

**回答**: ✅ **1時間以内で完了する。在庫もその日までに用意する**

**設計決定**:
- 標準: 1時間以内で完了（即日完了）
- 予約時にオイルフィルターの有無も確認済み
- 在庫もその日までに用意済み

**質問2**: エンジンオイル交換の費用はいくらですか？

**回答**: ✅ **添付画像の割引前の価格（¥10,560）**

**設計決定**:
- エンジンオイル交換(下抜き): ¥10,560（割引前）
- SM規格全合成油3L使用+レベル点検
- 法定費用はなし（オイル代・工賃のみ）

### 2-2. 検査項目について

**質問3**: エンジンオイル交換時の簡易検査項目は、どのように入力しますか？

**回答**: ✅ **必要であれば提案した項目を採用**

**推奨検査項目（提案）**:
1. **オイルの状態確認**
   - オイルの色・汚れ具合
   - オイル量（レベル点検）
2. **オイルフィルターの確認**
   - フィルターの状態
   - 交換の有無（予約時に確認済み）
3. **エアフィルターの確認**
   - フィルターの汚れ具合
   - 交換の必要性

**設計決定**:
- 簡易検査項目をリスト形式で表示
- 各項目の状態を選択（正常/注意/要交換）
- 必要に応じて写真撮影

**質問4**: 測定値の入力は必要ですか？

**回答**: ✅ **業務実態を把握してベストを提案**

**推奨案（業務実態を考慮）**:
- **測定値は基本的に不要**（簡易検査のみ）
- **オイル量の確認**: 「レベル点検」として、正常/不足/過多の選択で十分
- **オイルの粘度など**: 数値入力は不要（オイルの種類で判断）

**設計決定**:
- 測定値入力画面は不要
- オイル量は「レベル点検」として状態選択のみ

### 2-3. 写真・動画について

**質問6**: 写真・動画の撮影は必要ですか？

**回答**: ✅ **抜いたオイルなどあったほうがいい。どれだけ黒くなったかわかる**

**設計決定**:
- **Before写真**: 抜いたオイルの状態を撮影（どれだけ黒くなったか）
- **After写真**: 新しいオイルを入れた状態を撮影
- 動画は不要（写真のみ）
- 顧客への説明材料として有効

### 2-4. 部品リストアップについて

**質問7**: 診断時の部品リストアップは必要ですか？

**回答**: ✅ **追加の部品アップは基本は必要ない。イレギュラーで何か見つけた場合などはあるが基本はない**

**設計決定**:
- **基本**: 交換する部品は固定（オイル、オイルフィルター）で、リストアップは不要
- **イレギュラー**: 診断時に追加で交換が必要な部品が見つかった場合のみ、リストアップ機能を使用
- 診断画面には部品リストアップセクションを表示するが、通常は使用しない

### 2-5. 見積もりについて

**質問8**: 見積もりの承認フローはどうしますか？

**回答**: ✅ **見積りフローはいらない。そもそも作業の予約時に実施が確定しているので**

**重要な気づき**: 
- **パターンA（入庫してから見積もり）**: 入庫→見積もり→承認→作業（車検・12ヶ月点検など）
- **パターンB（事前にメール等で見積もり）**: メール等で見積もり→了承→入庫→作業（エンジンオイル交換、バッテリー交換など）

**設計決定**:
- エンジンオイル交換は**パターンB**（事前にメール等で見積もり→了承→入庫→作業）
- 見積もり画面は不要（予約時に実施が確定している）
- 承認フローも不要（事前に了承済み）

**質問9**: 作業記録はどうすればいいかわかりません。

**回答**: ✅ **要確認・提案**

**推奨案（提案）**:
- **作業記録**: 交換したオイルの種類・量、交換部品（オイルフィルター）を記録
- **Before/After写真**: 抜いたオイルと新しいオイルの写真
- **作業完了報告**: 顧客へのレポート画面で表示

**設計決定**:
- 作業画面で作業記録を入力
- 交換したオイルの種類・量を記録
- 交換部品（オイルフィルター）を記録
- Before/After写真を撮影

### 2-6. PDFテンプレートについて

**質問10**: エンジンオイル交換用のPDFテンプレートはありますか？

**回答**: ✅ **要確認**

**設計決定**:
- PDFテンプレートの有無を確認
- 必要に応じてPDFを生成

### 2-7. データソースについて

**質問11**: 車両情報や費用情報の取得元は車検・12ヶ月点検と同じですか？

**回答**: ✅ **データソースは同じグーグル**

**設計決定**:
- Google Sheets「【DB】車両マスタ」から車両情報を取得
- オイル代・工賃は固定価格（¥10,560）または基幹システムから取得

## 3. 確定した業務フロー

### 3-1. パターンA: 入庫してから見積もり（車検・12ヶ月点検など）

```
1. 顧客が来店（または予約）
2. フロントスタッフが受付
3. 整備士が診断を実施
4. 見積もりを作成
5. 顧客が承認
6. 作業を実施
7. 出庫完了
```

### 3-2. パターンB: 事前にメール等で見積もり（エンジンオイル交換など）

```
1. 顧客が予約（メール等で見積もりを送信）
2. フロントスタッフが見積もりを作成・送信
   - オイル代・工賃: ¥10,560
   - オイルフィルターの有無を確認済み
3. 顧客が了承（メール等で返信）
4. 入庫日を確定
5. 入庫当日
   - フロントスタッフが受付
   - 整備士が簡易検査を実施
     - オイルの状態確認
     - オイルフィルターの確認
     - エアフィルターの確認
     - 抜いたオイルの写真を撮影（Before）
   - 作業を実施
     - エンジンオイル交換
     - オイルフィルター交換
     - 新しいオイルの写真を撮影（After）
     - 作業記録を入力
   - 作業完了（1時間以内）
   - フロントスタッフが引渡し・説明
6. 出庫完了
```

### 3-3. 作業記録の内容

**記録項目**:
- 交換したオイルの種類（例: SM規格全合成油）
- 交換したオイルの量（例: 3L）
- 交換方法（例: 下抜き）
- 交換部品（オイルフィルター）
- Before/After写真（抜いたオイル、新しいオイル）
- 簡易検査結果（オイルの状態、オイルフィルター、エアフィルター）
