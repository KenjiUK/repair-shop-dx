# タイヤ交換・ローテーション 業務理解メモ

## ドキュメント情報

- **作成日**: 2025-01-XX
- **バージョン**: 1.0
- **対象**: タイヤ交換・ローテーションの業務理解
- **目的**: エンジンオイル交換と同様に、タイヤ交換・ローテーションの業務フローを想像し、仕様を確定する

---

## 1. タイヤ交換・ローテーションの基本情報

### 1-1. 業務の特徴

- **タイヤ交換**: 新しいタイヤに交換する作業
- **タイヤローテーション**: 既存のタイヤの位置を入れ替える作業（前後、左右）
- **頻度**: 定期的なメンテナンス（走行距離や期間に応じて）
- **作業内容**: タイヤの交換、バランス調整、空気圧調整
- **簡易検査**: タイヤの状態確認、溝の深さ確認、空気圧確認

### 1-2. 業務フローの特徴（推測）

- **タイヤ交換**: 新しいタイヤが必要なため、事前にタイヤを用意する必要がある
- **タイヤローテーション**: 既存のタイヤを使用するため、即日対応可能
- **承認フロー**: タイヤ交換は高額なため、事前承認が必要な可能性が高い
- **記録内容**: 交換したタイヤの種類・サイズ、ローテーション位置、バランス調整結果

### 1-3. オプションメニューとの関係

- **オプションメニュー**: 車検・12ヶ月点検と同時実施で10%割引の「タイヤ交換作業（4本）」
  - 通常価格: ¥30,800 → 割引価格: ¥27,720
  - 内容: バランス調整込(16インチ迄、タイヤ代別)
  - バッジ: 安全走行の要

## 2. 確認済み事項

### 2-1. 業務フローについて

**質問1**: タイヤ交換とローテーションは、同じ作業として扱いますか？それとも別々の作業として扱いますか？

**回答**: ✅ **別々の作業として扱う**

**質問2**: タイヤ交換・ローテーションの所要時間はどのくらいですか？

**回答**: ✅ **2時間**

**質問3**: タイヤ交換とローテーションの費用はいくらですか？

**回答**: ✅ 
- **タイヤ交換（4本）**: ¥30,800（バランス調整込、16インチ迄、タイヤ代別）
- **タイヤローテーション**: 工賃のみ（バランス調整込、16インチ迄、タイヤ代別）- 金額は「あたりでいれておいて」

**質問4**: タイヤ交換・ローテーションの業務フローパターンはどちらですか？

**回答**: ✅ **パターンB（事前にメール等で見積もり→了承→入庫→作業）。またタイヤ持ち込みの場合もある**

### 2-2. 検査項目について

**質問5**: タイヤ交換・ローテーション時の簡易検査項目は、どのように入力しますか？

**回答**: ✅ **特にないので輸入車の文脈の推奨でベストを提案**

**推奨検査項目（提案・採用）**:
1. **タイヤの状態確認**
   - タイヤの溝の深さ（前後左右）
   - タイヤの摩耗状態（偏摩耗、片減りなど）
   - タイヤの損傷（キズ、パンク痕、バルジなど）
   - タイヤの製造年（DOT表示）
2. **空気圧の確認**
   - 前後左右の空気圧
   - 推奨空気圧との比較
   - 空気圧の均一性
3. **ホイールの状態確認**
   - ホイールの損傷（クラック、曲がりなど）
   - ホイールナットの緩み
   - ホイールバランスの状態

**質問6**: 測定値の入力は必要ですか？

**回答**: ✅ **特にないので輸入車の文脈の推奨でベストを提案**

**推奨案（業務実態を考慮・輸入車の文脈）**: ✅ **採用**
- **タイヤ溝深さ**: 前後左右の溝深さを測定・記録（mm単位）
  - 法定基準: 1.6mm以上
  - 推奨基準: 3mm以上（輸入車の高性能タイヤの場合）
- **空気圧**: 前後左右の空気圧を測定・記録（kPaまたはpsi単位）
  - 推奨空気圧は車両情報から自動取得
  - 実際の空気圧と推奨空気圧を比較表示

### 2-3. 写真・動画について

**質問7**: 写真・動画の撮影は必要ですか？

**回答**: ✅ **はい**

**設計決定**:
- **Before写真**: 交換前のタイヤの状態（タイヤ交換の場合）またはローテーション前のタイヤ位置（ローテーションの場合）
- **After写真**: 交換後のタイヤの状態（タイヤ交換の場合）またはローテーション後のタイヤ位置（ローテーションの場合）
- 動画は不要（写真のみ）

### 2-4. 部品リストアップについて

**質問8**: 診断時の部品リストアップは必要ですか？

**回答**: ✅ **記録は費用だが部品リストアップは不要**

**設計決定**:
- **タイヤ交換**: 記録は費用だが部品リストアップ機能は不要（予約時に既に確定）
- **タイヤローテーション**: 部品リストアップは不要

### 2-5. 見積もり・承認フローについて

**質問9**: 見積もりの承認フローはどうしますか？

**回答**: ✅ **事前に決まりますね。他のメンテも事前に価格が決まってるので承認不要**

**設計決定**:
- 見積もり画面は不要（予約時に実施が確定している）
- 承認フローも不要（事前にメール等で了承済み）

### 2-6. 作業記録について

**質問10**: 作業記録はどうすればいいですか？

**回答**: ✅ **特にないので輸入車の文脈の推奨でベストを提案**

**推奨案（提案・採用）**:
- **タイヤ交換の場合**:
  - 交換したタイヤの種類・サイズ・メーカー
  - 交換位置（前後左右）
  - バランス調整結果（各ホイールのバランス値）
  - 空気圧設定値（前後左右）
  - Before/After写真
  - タイヤ溝深さ（交換前の記録）
- **タイヤローテーションの場合**:
  - ローテーション位置（例: 前左→後右、前右→後左、後左→前左、後右→前右）
  - バランス調整結果（各ホイールのバランス値）
  - 空気圧設定値（前後左右）
  - Before/After写真
  - タイヤ溝深さ（ローテーション前後の記録）

### 2-7. PDFテンプレートについて

**質問11**: タイヤ交換・ローテーション用のPDFテンプレートはありますか？

**回答**: ✅ **PDFテンプレートはない。PDF生成は不要**

**設計決定**:
- PDF生成は不要（エンジンオイル交換と同様）
- PDFで出すのは車検と12ヶ月点検のみ（決まったフォーマット）

### 2-8. データソースについて

**質問12**: 車両情報や費用情報の取得元は車検・12ヶ月点検と同じですか？

**回答**: ✅ **全部一緒です**

**設計決定**:
- Google Sheets「【DB】車両マスタ」から車両情報を取得
- 工賃は固定価格または基幹システムから取得

## 3. 業務フローの推測

### 3-1. タイヤ交換（パターンBの可能性が高い）

```
1. 顧客が予約（メール等で見積もりを送信）
2. フロントスタッフが見積もりを作成・送信
   - タイヤ代（顧客が選択したタイヤ）
   - 工賃: ¥30,800（16インチ迄、バランス調整込）
   - タイヤの種類・サイズを確認
3. 顧客が了承（メール等で返信）
4. タイヤを発注・用意
5. 入庫日を確定
6. 入庫当日
   - フロントスタッフが受付
   - 整備士が簡易検査を実施
     - タイヤの状態確認
     - 空気圧の確認
     - 交換前のタイヤの写真を撮影（Before）
   - 作業を実施
     - タイヤ交換（4本）
     - バランス調整
     - 空気圧調整
     - 交換後のタイヤの写真を撮影（After）
     - 作業記録を入力
   - 作業完了
   - フロントスタッフが引渡し・説明
7. 出庫完了
```

### 3-2. タイヤローテーション（パターンBの可能性が高い）

```
1. 顧客が予約（メール等で見積もりを送信）
2. フロントスタッフが見積もりを作成・送信
   - 工賃: ¥X,XXX（ローテーション工賃）
3. 顧客が了承（メール等で返信）
4. 入庫日を確定
5. 入庫当日
   - フロントスタッフが受付
   - 整備士が簡易検査を実施
     - タイヤの状態確認
     - 空気圧の確認
     - ローテーション前のタイヤの写真を撮影（Before）
   - 作業を実施
     - タイヤローテーション（位置変更）
     - バランス調整
     - 空気圧調整
     - ローテーション後のタイヤの写真を撮影（After）
     - 作業記録を入力
   - 作業完了
   - フロントスタッフが引渡し・説明
6. 出庫完了
```

## 5. 確定した仕様サマリー

### 5-1. 業務フロー

- **パターン**: パターンB（事前にメール等で見積もり→了承→入庫→作業）
- **所要時間**: 2時間で完了
- **タイヤ持ち込み**: 顧客が自分でタイヤを用意する場合もある

### 5-2. 費用

- **タイヤ交換（4本）**: ¥30,800（工賃のみ、タイヤ代別）
  - バランス調整込
  - 16インチ迄
- **タイヤローテーション**: 工賃のみ（金額は要確認・設定）
  - バランス調整込
  - 16インチ迄

### 5-3. 検査項目（採用）

1. タイヤの状態確認（溝深さ、摩耗状態、損傷、製造年）
2. 空気圧の確認（前後左右、推奨空気圧との比較）
3. ホイールの状態確認（損傷、ナットの緩み、バランス）

### 5-4. 測定値

- **必要**: 測定値入力画面を実装
- **タイヤ溝深さ**: 前後左右（mm単位）
- **空気圧**: 前後左右（kPaまたはpsi単位）

### 5-5. 写真撮影

- **Before写真**: 交換前のタイヤ（タイヤ交換）またはローテーション前の位置（ローテーション）
- **After写真**: 交換後のタイヤ（タイヤ交換）またはローテーション後の位置（ローテーション）

### 5-6. 部品リストアップ

- **タイヤ交換**: 記録は費用だが部品リストアップ機能は不要
- **タイヤローテーション**: 不要

### 5-7. 見積もり・承認フロー

- **見積もりフロー**: 不要（予約時に実施が確定）
- **承認フロー**: 不要（事前に了承済み）

### 5-8. 作業記録

- **タイヤ交換**: 交換したタイヤの種類・サイズ・メーカー、交換位置、バランス調整結果、空気圧設定値、Before/After写真、タイヤ溝深さ
- **タイヤローテーション**: ローテーション位置、バランス調整結果、空気圧設定値、Before/After写真、タイヤ溝深さ

### 5-9. データソース

- **車両情報**: Google Sheets「【DB】車両マスタ」から取得
- **費用**: 固定価格または基幹システムから取得

### 5-10. PDF生成

- **PDF生成**: 不要（タイヤ交換・ローテーションではPDF生成は行わない）
